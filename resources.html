Github

Github is a widely-used website and application that allows mutliple people to collaborate on one collection (or repository) of code. It also hosts websites for free.

HTML Text Editors

Sublime Text and Notepad++ are both good applications for writing code. I use Sublime Text 2 to edit and view HTML files. You can download them online.

Code Tutorials and Help

W3Schools has extremely helpful tutorials and quick explanations about HTML, CSS, jQuery, and other languages. Codeacademy is an in-depth tutorial site, good for if you're completely self-teaching a language. For specific issues, Stack Overflow has answers to basically every coding bug or question (and you can ask your own).

Bootstrap

Bootstrap is a simple (and pretty) framework for web development. I don't know much about how it works except that there are plenty of Bootstrap templates online that are easy enough to download and edit to fit my own projects.
